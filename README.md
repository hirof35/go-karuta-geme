# 本格派！Go言語いろはかるた (Karuta Game in Go)

Go言語の2Dゲームエンジン **Ebitengine** を使用して作られた、シンプルで爽快な「いろはかるた（ことわざカルタ）」ゲームです。
Windowsのシステムフォント環境に依存せず、フォントをプログラム内部に埋め込むことで、どの環境でも一撃で安定して起動する設計になっています。

## 特徴
- **ランダム配置**: 起動するたびに取り札の位置がシャッフルされます。
- **ミリ秒単位のタイムアタック**: 全ての札を取り終えるまでのタイムを競うことができます。
- **美しい日本語表示**: Google Fontsの「Noto Sans JP」を内包し、アンチエイリアスの効いた綺麗な文字を表示します。

## 動作環境
- Go 1.16 以上 (embed機能を使用するため)
- OS: Windows / macOS / Linux (Ebitengineがサポートする環境)

## フォルダ構成
```text
karuta-game/
├── main.go            # ゲームのソースコード
├── static/
│   └── NotoSansJP-Regular.ttf  # 日本語フォントファイル
└── README.md          # この説明ファイル
起動方法
リポジトリをクローンするか、ソースコードをフォルダに配置します。

static フォルダに NotoSansJP-Regular.ttf があることを確認します。

ターミナル（PowerShellやコマンドプロンプト）を開き、以下のコマンドを実行します。

Bash
go run main.go
遊び方
ゲームが起動すると、画面上部に「【読み札】」としてことわざが表示されます。

画面に並んだ白い取り札の中から、そのことわざの「頭文字（ひらがな1文字）」が書かれた札を探してください。

正しい札をマウスで 左クリック すると、その札を獲得できます。

全ての札（46枚）を取り終えるとゲームクリアとなり、クリアタイムが表示されます。

技術的ポイント
go:embed によるアセット内包: フォントファイルをバイナリとして実行ファイルに直接埋め込んでいるため、配布時にフォントの読み込みパスエラーが起きません。

TTC制限の回避: Ebitengineの text/v2 パッケージで発生しやすい「TTC（TrueType Collection）非対応エラー」を避けるため、単体の .ttf ファイルを明示的に指定して処理しています。
